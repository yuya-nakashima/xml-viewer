# XML Viewer

XMLファイルをブラウザで表示するためのビューアです。

## 機能

- XMLファイルの読み込み
- XSLファイルの読み込み（任意）
- ドラッグ＆ドロップ対応
- XMLの整形表示
- XMLのツリー表示
- XSL適用結果の表示
- 表示内容のコピー
- クリア機能

## 使い方

1. XMLファイルを選択します
2. 必要ならXSLファイルも選択します
3. 「表示」を押します
4. タブを切り替えて表示を確認できます

## ファイル構成

- `index.html`
- `style.css`
- `script.js`

## 補足

- XMLだけでも利用できます
- XSLがある場合は、XSL適用結果も表示できます
- 一部の複雑なXSLはブラウザ実装差異により正しく動かない場合があります
