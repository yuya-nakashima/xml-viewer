# XMLビューア

XMLファイルとXSLファイルをまとめて読み込み、帳票を表示するためのシンプルなビューアです。

## 使い方

1. XMLファイルとXSLファイルを複数選択します
2. または画面にまとめてドラッグ＆ドロップします
3. 同名のXMLファイルとXSLファイルが自動で組み合わされます
4. タブを切り替えて表示できます

## 仕様

- `a.xml` と `a.xsl` のような同名ファイルを自動でペアリングします
- XSLファイルがない場合は、手動でXSLを選択するか、XMLをそのまま表示できます
- GitHub Pagesで動作する純粋なHTML + JavaScriptのツールです
